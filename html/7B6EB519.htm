      Remarks

      Use this class to define, run and retrieve a result of your calculations.

      Calculation

      Class inheriting from EvaluationScope<T> is logical unit to compute desired result. It will containt lambda expressions,
      input parameters or constants of types derived from Value or IValue.

      Examples

      See below a class inheriting from EvaluationScope<T>, where T is a result class type.

      C#
      public class ExampleCalculation : EvaluationScope<Number>
{
public ExampleCalculation() : base(new EvaluationOptions { AlwaysReadNamesFromExpressions = true }) { }
public override Number Return() => Number.Of(0);
}
